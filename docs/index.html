Quick Start

# Install
pip install sacred-composer

# Compose
from sacred_composer import *

piece = CompositionBuilder(key="D_minor", tempo=72, bars=32) \
    .melody(pattern="fibonacci") \
    .bass() \
    .build()

piece.render("fibonacci_music.mid")

Features

Harmony Engine

Four-part voice leading, secondary dominants, modal interchange, cadence detection.

World Music

7 tuning systems: Western 12-TET, just intonation, Pythagorean, Arabic maqam, Indian raga, gamelan slendro, and pelog.

Adaptive Game Music

Real-time parameter control for tension, energy, and mood. Seamless transitions.

Consciousness Presets

Binaural beats, isochronic tones, and entrainment patterns for focus, relaxation, and meditation.

Psychoacoustics

Shepard tones, tritone paradox, missing fundamental, and auditory illusions as compositional tools.

Orchestration

Multi-instrument scoring with range-aware voice assignment and idiomatic writing.

REST API

FastAPI server for on-demand generation. Send parameters, receive MIDI or audio.

Discord Bot

Generate and share compositions directly in Discord channels.

Visualization

Piano roll, waveform, spectrogram, and pattern-space plots for every composition.
